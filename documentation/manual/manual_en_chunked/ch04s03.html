4.3. Virtual Keyboard

For quick checks and recording (or in case you do not have a MIDI device at hand), you can also use Hydrogen's Virtual Keyboard. It emulates note triggering via MIDI NOTE_ON events (see MIDI Note Rendering) but currently only supports order-based mapping in Input Mode Drumkit.

Note

Key strokes in the Virtual Keyboard do also not trigger Actions associated with NOTE_ON events.
